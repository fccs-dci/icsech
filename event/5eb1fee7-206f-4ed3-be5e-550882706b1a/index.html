Participants 嘉賓

主讲人 : 刘蟾
又名大生，青年作家、书法家。不参加国展，非书协会员。曾主编《中国书房》。出版《重新发现诗经》《诗词中的田园耕读》《名士的反抗: 从孔融到嵇康》等。2020年以来以书法记录民生疾苦、介入现实批判，创作诸多时事对联及作品。

评议人 : 周濂
中国人民大学哲学院教授，外国哲学教研室主任，中华全国外国哲学史学会理事；中国现代外国哲学学会政治哲学专业委员会副理事长。曾任牛津大学哲学系访问学者、哈佛燕京访问学者。 著有《现代政治的正当性基础》《你永远都无法叫醒一个装睡的人》《正义的可能》《正义与幸福》《打开 :周濂的100堂西方哲学课》，主编《西方政治哲学史·第三卷》在学术期刊发表中英文论文50余篇。研究领域包括政治哲学道德哲学和语言哲学。

主持人: 马国川
新人文学者，主要关注宏观经济、当代中国改革、日本近现代史等。创办并主持知识类高端视频直播节目《马上开谈》，先后任《经济观察报》高级记者，《财经》杂志主笔。著有《我与八十年代》《重启改革议程》(与吴敬琏先生合著)，《日本三部曲》《国家的抉择》等，荣获"文津图书奖"，多部作品译为英文、日文、韩文在海外发行。

Description 詳情

通过对汉未魏晋时期知识分子与权力的关系分析，讲者总结出了五种模态：建构式批判、解构式反抗、迎合权力、合作图存，以及自我内心的超越和突围。这五种模态，恰恰可以和汉未魏晋时期，形成某种巧妙对应的关系。这对当代知识群体如何进行价值判断、如何自处、如何选择提供了某种程度的借鉴。
