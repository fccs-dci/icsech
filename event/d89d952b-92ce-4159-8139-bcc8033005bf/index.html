Participants 嘉賓

主讲人： 路江涌
北京大学光华管理学院组织与战略管理系教授，北京大学博雅特聘教授。先后入选“教育部长江青年学者”和“教育部长江特聘教授”计划，获“国家自然科学杰出青年基金”资助，2020-2022年入选爱思唯尔中国高被引学者榜单，2019-2022年入选全球前2%顶尖科学家“年度影响力”榜单。研究专注于创新创业、企业成长、企业生态、组织协同、公司创投。出版《共演战略》《共演战略观》《共演战略画布》《危机共存》《图解创新管理经典》《图解企业成长经典》《科幻中的战略思维》《给企业的桌游战略课》等图书。

评议人兼主持人：周中华
现任北京颉腾文化传媒有限公司董事长，具有丰富的出版与数字内容服务经验。自2005年起，周中华先生曾担任华章公司董事总经理，并在2016年至2020年间，担任机工业出版副社长，兼任互动出版网络副董事长等职务。他不仅是著名的出版行业专家，还是多个文化和创新项目的创始人与投资人。

Description 詳情

人工智能时代带来的不仅是福祉，也有焦虑。作为碳基生命，人类应如何应对国际文明带来的挑战？路江涌教授从人工智能带来的挑战入手，以“觉而时行知”为理念，以“桌面游戏”为媒介，构造了一个适合企业和家庭的互动学习体系。通过对桌面游戏的系统研究和测试，路江涌教授从中选出了近百个系列桌游，并从管理学和教育学的角度入手，整理出“给企业的桌游管理课”和“给家庭的桌游亲子课”系列游戏，前者包括战略、组织、创新等系列，后者按孩子年龄分为4-6岁、7-9岁、10-12岁、13岁+等系列。系列丛书即将由颉腾文化出版，特邀请颉腾文化董事长周中华先生和路江涌交流共谈桌游、教育和管理，让各位网友在与家人共度的桌游时光中有更多收获。搜索“大学沙龙”YouTube频道收看讲座。
