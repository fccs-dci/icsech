Description 詳情

文明的两端是指人类文明开始的一端和现代的一端。戴蒙德等学者对文明演变持一种环境决定论的解释，但一种不同时段的多因论的解释看来更为合理，具体说来是一种“物质是基础、价值是主导、政治是关键”的逻辑。由此，一万余年的人类文明历史也呈现出一种“同——异——同”的面貌。我们可以考察这主导价值追求基本相同的两端，但是，文明首端物质追求的意义还是在奠定政治文明和精神文明的基础，而现代的一端则是以技术和经济的追求作为最高的目的。近代以来主导价值不断朝向提升控物能力和物质生活水平的转变，这和一种“平等的狡计”不无关系。文明会不会以物质始而以物欲终？搜索“大学沙龙”YouTube频道收看讲座。
