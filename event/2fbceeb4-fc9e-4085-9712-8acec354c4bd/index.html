Description 詳情

种姓歧视是印度文化中的历史性顽疾。印度独立运动领袖和制宪会议代表都赞成它是宪法亟需解决的核心议题之一，但对于如何解决却存在严重分歧。种姓特留权，作为妥协的结果，成为宪法性平权的核心制度之一。它对印度选举票仓有很大的摇摆力，也因此成为政客们最易开给选民们的政治支票。印度最高法院成为了该制度发展的关键平衡者，与联邦议会和邦议会不断拉锯，试图抑制政客们对这一制度的过度滥用与狂热扩张。  
种姓特留权问题是，自90年代经济自由化之后至今，印度的核心政治议题和最活跃宪法议题之一。其引发的进步、张力和纠结，也是印度独立后在传统宗教文化基础上嫁接现代民主自由制度所产生的惊喜、张力和纠结的一个缩影。本次讲座将聚焦于种姓特留权制度的产生及其运行，并力争呈现其实践纠结的核心逻辑。搜索“大学沙龙"YouTube 频道收看讲座。
