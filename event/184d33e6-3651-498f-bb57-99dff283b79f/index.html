Participants 嘉賓

主讲/主持人：邱泽奇。北京大学博雅特聘教授，教育部长江学者特聘教授。1994年获北京大学社会学博士。先后担任北京大学中国社会科学调查中心创始主任、重庆大学公共管理学院首任院长、中国质量奖评审委员会专家委员会副主任委员、阿里研究院学术委员会主席等职务。现任北京大学中国社会与发展研究中心主任、数字治理研究中心主任，兼任山西大学政治与公共管理学院院长、乡村振兴研究院院长，重庆大学公共管理学院名誉院长。2006年受北京大学委托创立中国家庭跟踪调查(CFPS)，担任PI组组长。重庆大学“费孝通勤学奖”创立者。先后获得过全国青年优秀读物一等奖、国家图书奖提名奖、首届国家图书馆文津图书奖、中国出版政府奖图书奖提名奖、高等学校科学研究优秀成果(人文社会科学)奖、北京大学教学成就奖等奖项。

主讲/评议人：甘犁。西南财经大学中国家庭金融调查与研究中心主任，美国德州A&M大学Clifford Taylor Jr. 经济学讲席教授。1987年获清华大学经济与管理学院获得学士，1998年获美国加州大学伯克利分校统计学硕士和经济学博士。发表了大量中英文中国经济和应用微观经济学领域的学术论文。2009年在西南财经大学发起并领导中国家庭金融调查(CHFS)，每两年进行一次，持续跟踪中国4万个有代表性的家庭，目前已经完成了六轮入户调查。CHFS数据库是国内外研究中国家庭经济行为的主要微观基础库之一。

主讲/评议人：王卫东。中国人民大学社会学系副教授、社会心理学研究所所长、中国调查与数据中心副主任。世界民意研究协会(WAPOR)中国代表，国际社会调查合作项目(ISSP)中国协调人，东亚社会调查(EASS)项目中国代表，中国教育追踪调查(CEPS)项目负责人，中国人宗教生活调查(CRLS)项目负责人，中国宗教调查(CRS)联合负责人，中国综合社会调查(CGSS)项目共同负责人，跨平台面访调查软件OmniSurvey及云电话调查软件OmniCATI的作者。研究领域主要为社会心理学、 健康社会学、 宗教社会学、教育社会学、社会网络分析、社会调查研究方法。

Description 詳情

二十一世纪初年，为推动中国学术研究规范化提供可靠数据，一些高校利用国家财政的学科建设经费发起了主题化大规模概率抽样调查。其中，以中国人民大学的中国社会综合调查(CGSS)、北京大学的中国家庭追踪调查(CFPS)、北京大学的中国健康与养老追踪调查(CHARLS)、西南财经大学的中国家庭金融调查(CHFS)最具影响力。如今，这些调查的先行者已经二十年。CGSS、CFPS、CHFS的主导者和亲历者将向各位讲述项目的缘起、发展、影响，以及在数字时代面临的挑战。请搜索“大学沙龙”YouTube频道收看当期及往期讲座。
