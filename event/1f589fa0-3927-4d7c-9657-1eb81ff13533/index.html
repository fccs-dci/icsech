Participants 嘉賓

主讲人：陶然。香港中文大学(深圳)人文社科学院教授，公共政策研究部主任。先后获兰州大学学士、北京大学理学硕士、芝加哥大学经济学硕士和博士学位。曾在英国牛津大学中国研究所进行博士后研究。研究专注于中国经济转型与发展的多个相关领域，包括发展经济学、制度经济学、转型与增长的政治经济学逻辑、城市化进程中的土地和户籍改革、地方治理及公共财政等。主持过两项国家自然科学基金重点项目，正在主持一项国家社科基金重大项目，并在中国户籍、土地、住房改革等领域发挥了一定的政策影响。在国内外知名学术期刊发表学术论文112篇。2020年起连续三年入选爱思唯尔(Elsevier)发布的中国高引用学者榜的“理论经济学”榜单。

评议人：党国英。经济学博士。先后在陕西师范大学(77级)、兰州大学(85级)、中国社会科学院研究生院(94级)完成大学各期学业。在兰州大学长期担任经济学原理、经济思想史课程的教学工作。在中国社会科学院农村发展研究所从事研究工作直至退休，期间任研究室主任等。论文《农民政治参与的行为逻辑》(与胡冰川合作完成)获农村发展研究杜润生奖。出版《经济学理性》《中国农村改革》等著作。获中国社科院优秀科研人员荣誉奖(2010-2015年度)。主张通过土地制度市场化改革提高中国农业经济效率，通过均衡城市化政策支持中国农村发展。

主持人：张军。加拿大多伦多大学地理与城市规划系任副教授。美国明尼苏达大学地理系博士。主要研究领域为经济地理学、城市与区域经济发展、比较政治和制度经济学、全球化背景中的产业创新和演化。

Description 詳情

讲座将在对中国现有增长及工业化、城市化模式的形成和机制进行整体性考察的基础上，探讨中国城乡土地管理体制各个主要领域的政策演化及其局限，并探讨如何通过环环相扣的改革方案设计和政策组合来逐步扭转现有模式累积的各种扭曲，并最终推动环环良性的渐进式改革。这些解决方案都是在既有政策框架的基础上以逐步缓解现有矛盾、减少既有扭曲为目标的适度调整。既不是推倒重来，也非无关痛痒的修补，而是要通过一系列“激励相容”的联动改革逐步扭转现有土地制度引发的效率低下和分配不公问题。报告将针对集体经营性建设用地的入市，“成片开发”实施征地的门槛，政府征地补偿的模式及标准，以及农村土地整理、确权和农村土地所有制走向，各类土地指标的空间转移及跨区交易，城市低效存量用地再开发，城中村、城边村及老旧小区的更新改造等多个问题，提出与现有政策不同，甚至还充满张力的改革建议。搜索“大学沙龙"YouTube频道收看讲座。
