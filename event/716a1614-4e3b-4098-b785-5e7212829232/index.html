Participants 嘉賓

主讲人：郝景芳  小说作者，经济研究员，童行书院、芳景科幻创始人。2002年获第四届新概念作文大赛一等奖，2006年毕业于清华大学物理系，2013年清华大学经管学院博士毕业，2013年至2018年就职于中国发展研究基金会研究一部；2016年凭短篇小说《北京折叠》获第74届世界科幻大会雨果奖，被评选为《人物》《南方人物周刊》“十大年度人物”。2018年哈佛大学肯尼迪政府学院访问学者，同年被世界经济论坛评选为2018年“全球青年领袖”。主要著作有《孤独深处》《流浪苍穹》《人之彼岸》《去远方》《孩子，愿你一生勇敢，心中有光》《宇宙跃迁者》《中国前沿：不如问问科学家吧》等。

评议人：祁晓亮  斯坦福大学物理系教授，清华大学高等研究院博士，AI初创企业Path Integral Technology联合创始人。主要研究方向包括凝聚态物理，量子信息，量子引力和人工智能。曾获得物理学新视野奖等奖项。

主持人：段鸣镝  鸣镝科普基金会创始人。前中央电视台科教中心首席导演，最佳金鸡奖、华表奖科教片导演。中国视协纪录片学术委员会理事。中国传媒大学杰出校友，客座讲师。自媒体《老鸣TV》节目制作者，作品集《花与科学的彼岸》为今日头条科普公开课收视第3名。

Description 詳情

当前国际局势风云变幻、新兴科技带来社会变革，人类未来何去何从？科幻小说提供对于未来的思想实验，不一定成为现实，但一定可以提供逻辑严谨的、未来可能性的探讨。在科幻世界中，我们可以自由讨论AI对劳动力的取代、AI的社会影响、元宇宙的经济模式和对社会结构的冲击、国际战争对人类文明的打击、人类面向宇宙的希望与威胁，等等等等。探讨科幻文学的推演合理性，以及当前国际局势的未来思考。搜索“大学沙龙”YouTube频道收看讲座。
